## Project Overview
The Campus Guide Mobile App is a software engineering project designed to enhance the navigational experience within the Concordia University environment. Developed by team O(n)Campus, the application aims to provide a centralized, user-friendly mobile solution for campus navigation.
### Objectives
The primary goal of the project is to solve the complexity of navigating a large, multi-campus university. Many users face challenges when trying to locate specific buildings, classrooms, or services across both the SGW and Loyola campuses. All the information concerning the project are explained directly on the [Project Wiki](https://github.com/SOEN390OnCampus/OnCampusApp/wiki/01.-Introduction).

Download Android Studio Otter 2 Feature Drop (2025.2.2) from the official archive: https://developer.android.com/studio/archive

### 3. Open the Project

Open the cloned OnCampusApp project in Android Studio

### 4. Gradle Configuration

Ensure the project uses the following versions:

Android Gradle Plugin: 8.7
Gradle: compatible with AGP 8.7

Update Gradle files if needed, then re-sync the project.

<img width="947" height="503" alt="image" src="https://github.com/user-attachments/assets/6dc26e93-eced-4142-8dd1-39ab148c2a83" />

### 5. Java Version

Make sure you have Java JDK 17 or newer installed.

<img width="606" height="99" alt="image" src="https://github.com/user-attachments/assets/2d030744-f582-4e59-a0f3-308bc18ea4de" />

### 6. Google Maps API Key Setup

This project requires a Google Maps API key. To follow the steps to obtain the Maps API key, follow the developers guide made by Google : https://developers.google.com/maps/documentation/javascript/get-api-key 
Choose the free trial or no cost billing plan for the API key, then create a Google Maps API key from the Google Cloud Console.

<img width="1292" height="639" alt="image" src="https://github.com/user-attachments/assets/51faabd0-060a-4a2d-a6dd-85e6f26ea5a8" />

Open the local.properties file inside the project folder and add the following line: 

MAPS_API_KEY= YOUR_API_KEY_HERE
